# tutorial

Generate a complete Craft with Mommy single-craft tutorial blog article.

# Purpose

This skill is for Craft with Mommy tutorial articles focused on one specific craft only.

A tutorial article must teach one craft from start to finish in a warm, simple, beginner-friendly way. It should help a mom create a low-stress, low-mess, joyful bonding moment with her child.

Do not use this skill for roundup / ideas articles.

# Article Structure

Follow this tutorial article structure exactly.

This structure is mandatory for all single-craft tutorial articles.

## Top of Article Layout

At the very top of the article, use this order:

1. Breadcrumb
2. Category / age / time / messiness pills
3. Article title
4. Publication date
5. Hero image

## Breadcrumb Rules

The breadcrumb must always appear in this format:

Home › Blog › [Article Title]

Rules:

- Keep the breadcrumb on one line only.
- On mobile, the article title inside the breadcrumb must be truncated with an ellipsis if it is too long.
- Never allow the breadcrumb to wrap onto a second line.
- The truncation must keep the layout clean and compact on mobile.
- Do not let a long title push the layout downward.
- The breadcrumb must remain visually clean and compact.

## Top Pills Rules

Below the breadcrumb, display the tutorial metadata pills in this order:

1. Craft type / category
2. Age
3. Time
4. Messiness

Rules:

- These pills are required for tutorial articles.
- Estimate them realistically based on the actual craft.
- Age must usually fall between 2+ and 10+.
- Time must be realistic for the full activity, including setup and doing the craft.
- Time should usually be between 15 minutes and 1 hour 30 minutes maximum.
- If the full craft time is mentioned elsewhere in the article, it must stay consistent with the time shown in the top time pill.
- Messiness must always be one of:
  - Low
  - Medium
  - High
- These values must make sense for a real mom doing the craft with a young child.
- Do not guess carelessly.
- The pills must look intentional and coherent with the craft.

## Category Pill

Use the most accurate craft type for the article.

Typical examples include:

- Paper Crafts

- Recycled Crafts

# Image Generation and Image SEO Rules

All tutorial images must be generated with strong visual consistency, correct dimensions, SEO-friendly filenames, and a realistic handmade look.

## Required Image Dimensions

- Every article image should be requested from Gemini in 4:3 landscape format.
- The target size is 1200 pixels wide by 900 pixels high.
- The prompt should explicitly ask for a wide rectangular landscape image in 4:3 format.
- Gemini should be asked to generate the image as close as possible to 1200 × 900 from the start.
- If the returned image is not exactly 1200 × 900, do not stretch or squash it to fit
- Always preserve the original aspect ratio
- If needed, crop the image cleanly to a 4:3 landscape ratio before resizing
- After cropping, resize the final image to 1200 × 900
- Never flatten, distort, or compress the image unnaturally just to force the size

## File Format Rules

- Convert every final article image to WEBP.
- Keep the image lightweight and web-friendly.
- Do not insert PNG or JPEG directly into the final article when WEBP is available.
- The final saved image used in the article should be the WEBP version.

## Prompt Precision Rules

- Every image prompt must be highly specific.
- Do not use vague prompts.
- Clearly describe the craft, the materials, the color, the stage of the project, the atmosphere, and the composition.
- For tutorial articles, the prompt must describe the exact craft state for that exact image position.
- Avoid generic "kids crafting" prompts when the image needs to show a specific stage of the project.

## Step Image Continuity Rules

For step images, the craft must remain the same craft throughout the full tutorial.

This means:

- the same project must visually continue from step to step
- if something is added in step 2, it should still appear in step 3
- if the plate is painted green in one step, it should stay green in the next relevant step
- if a shape was cut out in an earlier step, that change must still be visible later
- the object must evolve logically across the steps
- the image sequence must show real progression, not disconnected scenes

Each step image must represent:

- the exact step described in the text
- the same craft project at the correct moment in its progression
- a visually coherent continuation of the previous step

## Hero and Secondary Image Rules

- The hero image must show the final completed craft only.
- It must match the final result described in the tutorial.
- The "Why Kids Love This Craft" image should ideally show a mother and child preparing the craft together at a craft table.
